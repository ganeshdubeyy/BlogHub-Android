╔══════════════════════════════════════════════════════════════╗
║          QUICK GIT COMMANDS - BlogHub GitHub Push           ║
╚══════════════════════════════════════════════════════════════╝

📋 COPY-PASTE THESE COMMANDS IN ORDER:

═══════════════════════════════════════════════════════════════
STEP 1: Navigate to Project
═══════════════════════════════════════════════════════════════
cd "d:\Cris Sir App\Android Project"


═══════════════════════════════════════════════════════════════
STEP 2: Initialize Git (First Time Only)
═══════════════════════════════════════════════════════════════
git init


═══════════════════════════════════════════════════════════════
STEP 3: Configure Git (First Time Only - Replace with YOUR info)
═══════════════════════════════════════════════════════════════
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"


═══════════════════════════════════════════════════════════════
STEP 4: Add All Files
═══════════════════════════════════════════════════════════════
git add .


═══════════════════════════════════════════════════════════════
STEP 5: Check Status (Optional - to verify)
═══════════════════════════════════════════════════════════════
git status


═══════════════════════════════════════════════════════════════
STEP 6: Create First Commit
═══════════════════════════════════════════════════════════════
git commit -m "Initial commit: BlogHub Android app with Jetpack Compose and Firebase"


═══════════════════════════════════════════════════════════════
STEP 7: Add Remote Repository (Replace YOUR_GITHUB_URL)
═══════════════════════════════════════════════════════════════
git remote add origin YOUR_GITHUB_URL

Example:
git remote add origin https://github.com/yourusername/BlogHub-Android.git


═══════════════════════════════════════════════════════════════
STEP 8: Rename Branch to 'main'
═══════════════════════════════════════════════════════════════
git branch -M main


═══════════════════════════════════════════════════════════════
STEP 9: Push to GitHub
═══════════════════════════════════════════════════════════════
git push -u origin main


═══════════════════════════════════════════════════════════════
